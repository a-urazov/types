# Пакет Matrix

Пакет `matrix` предоставляет универсальную реализацию двумерной матрицы с поддержкой различных операций.

## Особенности

- Обобщенная (generic) реализация, поддерживающая любые типы данных
- Поддержка числовых матриц с арифметическими операциями
- Основные операции: сложение, умножение, транспонирование
- Методы для доступа и модификации элементов
- Вспомогательные методы: клонирование, заполнение, сравнение

## Установка

```bash
# Пакет является частью модуля types
```

## Использование

### Создание матрицы

```go
// Создание новой матрицы размером 3x3
matrix, err := matrix.New[int](3, 3)
if err != nil {
    // обработка ошибки
}

// Создание матрицы с начальными значениями
values := [][]int{{1, 2, 3}, {4, 5, 6}, {7, 8, 9}}
matrix, err := matrix.NewWithValues(values)
if err != nil {
    // обработка ошибки
}
```

### Основные операции

```go
// Получение и установка значений
value, err := matrix.Get(0, 0)
if err != nil {
    // обработка ошибки
}

err = matrix.Set(0, 0, 42)
if err != nil {
    // обработка ошибки
}

// Получение размеров
rows := matrix.Rows()
cols := matrix.Cols()
r, c := matrix.Size()
```

## Типы

- `Matrix[T any]` - базовый тип матрицы для любого типа данных

## Методы

### Matrix[T]

- `New(rows, cols int)` - создает новую матрицу
- `NewWithValues(values [][]T)` - создает матрицу с заданными значениями
- `Rows()`, `Cols()` - возвращает размеры матрицы
- `Size()` - возвращает количество строк и столбцов
- `Get(row, col int)` - возвращает значение элемента
- `Set(row, col int, value T)` - устанавливает значение элемента
- `Clone()` - создает копию матрицы
- `Fill(value T)` - заполняет всю матрицу указанным значением
- `Reset()` - обнуляет матрицу
- `IsEqual(other *Matrix[T])` - сравнивает две матрицы
- `ForEach(fn func(T, int, int))` - применяет функцию к каждому элементу
- `Transpose()` - возвращает транспонированную матрицу
- `Apply(fn func(T, int, int) T)` - применяет функцию к каждому элементу, изменяя матрицу

## Ошибки

Методы могут возвращать ошибки в следующих случаях:
- Некорректные размеры матрицы (отрицательные или нулевые значения)
- Выход за пределы индексов при доступе к элементам
- Несовместимые размеры при выполнении операций (например, сложение матриц разных размеров)
