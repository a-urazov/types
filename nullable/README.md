# Обобщенный Nullable Тип для Go

Этот пакет предоставляет обобщенный nullable тип `Type[T any]`, который может содержать значение любого типа `T` или быть null. Это полезно для обработки необязательных значений в Go, особенно при работе с базами данных или API, где значения могут быть null.

## Возможности

- Обобщенная реализация, поддерживающая любой тип `T`
- Поддержка маршалинга/анмаршалинга JSON
- Интерфейсы драйвера базы данных (`driver.Valuer` и `sql.Scanner`)
- Методы для проверки, является ли значение null или нет
- Вспомогательные методы для установки значений и значений по умолчанию

## Использование

```go
package main

import (
    "fmt"
    "nullable"
)

func main() {
    // Создание nullable строки со значением
    name := nullable.New("John Doe")
    fmt.Printf("Имя: %s, Является Null: %t\n", name.String(), name.IsNull())
    
    // Создание nullable строки с null значением
    var nullName nullable.Type[string] = nullable.New[string]()
    fmt.Printf("Null Имя: %s, Является Null: %t\n", nullName.String(), nullName.IsNull())
    
    // Использование метода Or для предоставления значения по умолчанию
    defaultName := nullName.Or("Аноним")
    fmt.Printf("Имя по умолчанию: %s\n", defaultName)
    
    // Пример с целым числом
    age := nullable.New(25)
    fmt.Printf("Возраст: %s, Является Null: %t\n", age.String(), age.IsNull())
}
```

## API

### Type[T any]
Основной обобщенный nullable тип, который может содержать значение типа `T` или быть null.

### Методы

- `New[T any](value T) Type[T]` - Создает новый Type[T] со значением
- `IsNull() bool` - Проверяет, является ли Type[T] null
- `IsNotNull() bool` - Проверяет, есть ли у Type[T] значение
- `Set(value T)` - Устанавливает значение и помечает его как действительное
- `SetNull()` - Устанавливает значение в null
- `Get() (T, bool)` - Возвращает значение и булево значение, указывающее, является ли оно действительным
- `Or(defaultValue T) T` - Возвращает значение, если оно действительное, в противном случае возвращает предоставленное значение по умолчанию
- `String() string` - Возвращает строковое представление Type[T]

### Интерфейсы

`Type[T]` реализует:
- `json.Marshaler` - для маршалинга JSON
- `json.Unmarshaler` - для анмаршалинга JSON
- `driver.Valuer` - для операций с базой данных
- `sql.Scanner` - для операций с базой данных
