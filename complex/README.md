# Пакет Complex

Пакет `complex` предоставляет реализацию комплексных чисел с поддержкой различных математических операций.

## Особенности

- Реализация комплексных чисел с вещественной и мнимой частями
- Поддержка арифметических операций: сложение, вычитание, умножение, деление
- Методы для вычисления модуля, сопряжения, полярных координат
- Поддержка возведения в степень и извлечения квадратного корня

## Установка

```bash
# Пакет является частью модуля types
```

## Использование

### Создание комплексного числа

```go
// Создание нового комплексного числа 3 + 4i
c1 := complex.New(3, 4)

// Создание комплексного числа из полярных координат
c2 := complex.FromPolar(5, math.Pi/4) // модуль 5, угол π/4
```

### Арифметические операции

```go
c1 := complex.New(3, 4)
c2 := complex.New(1, 2)

// Сложение
sum := c1.Add(c2)           // (3+4i) + (1+2i) = 4+6i

// Вычитание
diff := c1.Subtract(c2)     // (3+4i) - (1+2i) = 2+2i

// Умножение
prod := c1.Multiply(c2)     // (3+4i) * (1+2i) = -5+10i

// Деление
quot := c1.Divide(c2)       // (3+4i) / (1+2i) = 2.2-0.4i
```

### Другие операции

```go
c := complex.New(3, 4)

// Модуль (абсолютное значение)
mag := c.Magnitude()        // sqrt(3² + 4²) = 5

// Сопряженное число
conj := c.Conjugate()       // 3-4i

// Полярные координаты
r, theta := c.Polar()       // модуль и угол

// Возведение в степень
power := c.Power(2)         // (3+4i)²

// Квадратный корень
sqrt := c.Sqrt()            // sqrt(3+4i)
```

## API

- `New(real, imag float64) Complex` - создает новое комплексное число
- `FromPolar(r, theta float64) Complex` - создает комплексное число из полярных координат
- `Add(other Complex) Complex` - сложение
- `Subtract(other Complex) Complex` - вычитание
- `Multiply(other Complex) Complex` - умножение
- `Divide(other Complex) Complex` - деление
- `Magnitude() float64` - модуль
- `Conjugate() Complex` - сопряженное число
- `Polar() (r, theta float64)` - полярные координаты
- `Power(n int) Complex` - возведение в степень
- `Sqrt() Complex` - квадратный корень
- `Equals(other Complex, tolerance float64) bool` - сравнение с учетом погрешности
- `String() string` - строковое представление
