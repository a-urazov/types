# Стек

Пакет `stack` предоставляет реализацию стека LIFO (Last In First Out). В стеке элементы добавляются и удаляются с одного конца (вершины), что означает, что последний добавленный элемент будет первым удален.

## Основные методы

- `New[T]() *Stack[T]` - создает новый пустой стек
- `Push(item T)` - добавляет элемент на вершину стека
- `Pop() (T, bool)` - удаляет и возвращает элемент с вершины
- `Peek() (T, bool)` - получает элемент с вершины без удаления
- `Size() int` - возвращает количество элементов
- `IsEmpty() bool` - проверяет, пуст ли стек
- `Clear()` - очищает стек

## Пример использования

```go
package main

import (
    "fmt"
    "types/collections/stack"
)

func main() {
    s := stack.New[string]()

    s.Push("first")
    s.Push("second")
    s.Push("third")

    for !s.IsEmpty() {
        if item, ok := s.Pop(); ok {
            fmt.Println("Popped:", item)
        }
    }
}
```

## Особенности

- **LIFO порядок**: последний пришедший элемент первым уходит
- **Потокобезопасность**: встроенная синхронизация
- **Простота**: минимальный набор методов для базовых операций

## Применение

- Отмена/повторение операций (undo/redo)
- Проверка скобок и выражений
- Глубинный поиск (DFS) в графах
- Вычисление выражений в обратной польской нотации
- Обработка вызовов функций (стек вызовов)
- Алгоритм обхода лабиринта
