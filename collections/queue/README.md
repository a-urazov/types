# Queue

Пакет `queue` предоставляет реализацию очереди FIFO (First In First Out). Это базовая структура данных, где элементы добавляются с конца и удаляются с начала, следуя принципу "первым пришел, первым ушел".

## Основные методы

- `New[T]() *Queue[T]` - создает новую пустую очередь
- `Enqueue(item T)` - добавляет элемент в конец очереди
- `Dequeue() (T, bool)` - удаляет и возвращает элемент с начала
- `Peek() (T, bool)` - получает элемент с начала без удаления
- `Size() int` - возвращает количество элементов
- `IsEmpty() bool` - проверяет, пуста ли очередь
- `Clear()` - очищает очередь

## Пример использования

```go
package main

import (
    "fmt"
    "types/collections/queue"
)

func main() {
    q := queue.New[string]()
    
    q.Enqueue("first")
    q.Enqueue("second")
    q.Enqueue("third")
    
    if item, ok := q.Dequeue(); ok {
        fmt.Println("Dequeued:", item)
    }
    
    fmt.Println("Size:", q.Size())
}
```

## Специализированные варианты

- **[Deque](./deque/)** - двусторонняя очередь (добавление и удаление с обоих концов)
- **[Priority Queue](./priority/)** - приоритетная очередь (элементы упорядочены по приоритету)
- **[Ring Buffer](./ring/)** - кольцевой буфер (фиксированный размер, циклическая структура)

## Применение

- Системы обработки задач
- Буферизация данных
- Симуляции реальных очередей
- Breadth-First Search (BFS) алгоритмы
