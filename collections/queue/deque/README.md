# Deque

Пакет `deque` предоставляет реализацию дека (Double-Ended Queue) - очереди с двумя концами. Deque позволяет эффективно добавлять и удалять элементы как с начала, так и с конца структуры.

## Основные методы

- `New[T]() *Deque[T]` - создает новый пустой дек
- `PushFront(item T)` - добавляет элемент в начало
- `PushBack(item T)` - добавляет элемент в конец
- `PopFront() (T, bool)` - удаляет и возвращает элемент с начала
- `PopBack() (T, bool)` - удаляет и возвращает элемент с конца
- `PeekFront() (T, bool)` - получает элемент с начала без удаления
- `PeekBack() (T, bool)` - получает элемент с конца без удаления
- `Size() int` - возвращает количество элементов
- `IsEmpty() bool` - проверяет, пуст ли дек
- `Clear()` - очищает дек

## Пример использования

```go
package main

import (
    "fmt"
    "types/collections/queue/deque"
)

func main() {
    d := deque.New[int]()
    
    d.PushFront(1)
    d.PushBack(2)
    d.PushBack(3)
    d.PushFront(0)
    
    if item, ok := d.PopFront(); ok {
        fmt.Println("Popped from front:", item)
    }
    
    fmt.Println("Size:", d.Size())
}
```

## Особенности

- **Двусторонняя навигация**: быстрые операции с обоих концов
- **Потокобезопасность**: встроенная синхронизация
- **Гибкость**: может использоваться как очередь или стек

## Применение

- Реализация окон скользящего размера
- Задачи "undo/redo" с ограничением
- Палиндромная проверка
- Планировщики задач с приоритизацией
