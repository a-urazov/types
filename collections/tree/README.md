# Tree

Пакет `tree` содержит реализации различных древовидных структур данных. Деревья используются для представления иерархических данных и обеспечения эффективного поиска, вставки и удаления.

## Доступные структуры деревьев

- **[Binary Search Tree (BST)](./bst/)** - двоичное дерево поиска для упорядоченного хранения данных
- **[B-Tree](./btree/)** - многоуровневое дерево поиска, оптимизированное для дисковых операций
- **[Trie](./trie/)** - префиксное дерево для хранения и поиска строк

## Общие концепции

- **Узлы и ребра**: деревья состоят из узлов, соединенных ребрами
- **Корень**: верхний узел дерева
- **Листья**: узлы без дочерних элементов
- **Высота**: длина самого длинного пути от корня к листу

## Пример использования

```go
package main

import (
    "fmt"
    "types/collections/tree/bst"
)

func main() {
    tree := bst.New[int]()
    
    tree.Insert(5)
    tree.Insert(3)
    tree.Insert(7)
    tree.Insert(2)
    tree.Insert(4)
    
    if tree.Search(3) {
        fmt.Println("3 found in tree")
    }
}
```

## Применение деревьев

- Поиск данных (Binary Search Tree)
- Файловые системы (B-Tree)
- Автодополнение (Trie)
- Синтаксический анализ
- Выражение вычислений
- Хранение иерархических данных
