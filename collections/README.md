# Collections

Пакет `collections` содержит реализацию различных обобщенных структур данных на Go. Каждая структура безопасна для использования в многопоточной среде благодаря встроенным мутексам.

## Доступные структуры данных

- **[List](./list/)** - универсальный список с методами добавления, вставки и удаления элементов
- **[Queue](./queue/)** - очередь FIFO (First In First Out) со специализированными вариантами:
  - **[Deque](./queue/deque/)** - дек (двусторонняя очередь)
  - **[Priority Queue](./queue/priority/)** - приоритетная очередь
  - **[Ring Buffer](./queue/ring/)** - кольцевой буфер
- **[Stack](./stack/)** - стек LIFO (Last In First Out)
- **[Set](./set/)** - набор уникальных элементов
- **[SortedSet](./sortedset/)** - отсортированное множество уникальных элементов
- **[BitSet](./bitset/)** - битовое множество для эффективного хранения целых чисел
- **[BloomFilter](./bloomfilter/)** - вероятностный фильтр для проверки принадлежности элемента множеству
- **[LRUCache](./lrucache/)** - кэш с политикой LRU (наименее недавно использованный) с автоматическим вытеснением
- **[DisjointSet](./disjointset/)** - система непересекающихся множеств (Union-Find) для отслеживания связности компонентов
- **[SegmentTree](./segmenttree/)** - дерево отрезков для эффективных запросов на диапазонах
- **[FenwickTree](./fenwicktree/)** - дерево Фенвика (BIT) для динамических префиксных сумм
- **[ConcurrentMap](./concurrentmap/)** - потокобезопасная мапа с оптимизированной производительностью через разделение блокировок
- **[Dictionary](./dictionary/)** - словарь (ключ-значение) со специализированными вариантами:
  - **[Sorted Dictionary](./dictionary/sorted/)** - отсортированный словарь
- **[Tree](./tree/)** - деревья со специализированными вариантами:
  - **[Binary Search Tree](./tree/bst/)** - двоичное дерево поиска
  - **[B-Tree](./tree/btree/)** - B-дерево
  - **[Trie](./tree/trie/)** - префиксное дерево

## Особенности

- **Обобщения (Generics)**: все структуры используют обобщения Go для типобезопасности
- **Потокобезопасность**: встроенная синхронизация (sync.RWMutex) для безопасной работы в многопоточной среде
- **Надежность**: полное покрытие модульными тестами

## Пример использования

```go
package main

import (
    "fmt"
    "types/collections/list"
)

func main() {
    // Создаем новый список целых чисел
    l := list.New[int]()
    
    // Добавляем элементы
    l.Add(10)
    l.Add(20)
    l.Add(30)
    
    // Итерируем по элементам
    l.ForEach(func(item int) {
        fmt.Println(item)
    })
}
```
