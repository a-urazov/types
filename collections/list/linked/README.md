# LinkedList

Пакет `linked` предоставляет реализацию двусвязного списка (doubly linked list). Каждый узел имеет ссылки как на предыдущий, так и на следующий элемент, что позволяет эффективно выполнять операции как с начала, так и с конца списка.

## Основные методы

- `New[T]() *List[T]` - создает новый пустой двусвязный список
- `AddFirst(value T)` - добавляет элемент в начало списка
- `AddLast(value T)` - добавляет элемент в конец списка
- `RemoveFirst() (T, bool)` - удаляет элемент с начала
- `RemoveLast() (T, bool)` - удаляет элемент с конца
- `GetFirst() (T, bool)` - получает элемент с начала
- `GetLast() (T, bool)` - получает элемент с конца
- `Size() int` - возвращает количество элементов
- `Clear()` - очищает список
- `Contains(value T) bool` - проверяет наличие элемента

## Пример использования

```go
package main

import (
    "fmt"
    "types/collections/list/linked"
)

func main() {
    list := linked.New[int]()
    
    list.AddFirst(1)
    list.AddLast(2)
    list.AddLast(3)
    
    if first, ok := list.GetFirst(); ok {
        fmt.Println("First:", first)
    }
    
    list.RemoveLast()
    fmt.Println("Size:", list.Size())
}
```

## Особенности

- **Двусторонняя навигация**: эффективный доступ как с начала, так и с конца
- **Потокобезопасность**: встроенная синхронизация для работы в многопоточной среде
- **Гибкость**: хорошо подходит для операций, требующих частого добавления/удаления с обоих концов
