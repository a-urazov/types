# SkipList

Пакет `skip` предоставляет реализацию списка с перепрыгиванием (Skip List). Skip List - это вероятностная структура данных, которая позволяет быстрый поиск в отсортированном списке с временной сложностью O(log n) в среднем случае, сохраняя при этом простоту связанного списка.

## Основные методы

- `New[T]() *List[T]` - создает новый пустой skip list
- `Insert(value T)` - вставляет значение в отсортированном порядке
- `Search(value T) bool` - ищет значение в списке
- `Delete(value T) bool` - удаляет значение из списка
- `Size() int` - возвращает количество элементов
- `DisplayList()` - выводит структуру списка (для отладки)

## Пример использования

```go
package main

import (
    "fmt"
    "types/collections/list/skip"
)

func main() {
    sl := skip.New[int]()
    
    sl.Insert(3)
    sl.Insert(1)
    sl.Insert(4)
    sl.Insert(1)
    sl.Insert(5)
    
    if sl.Search(3) {
        fmt.Println("Found 3")
    }
    
    sl.Delete(1)
    fmt.Println("Size:", sl.Size())
}
```

## Особенности

- **Быстрый поиск**: O(log n) в среднем случае
- **Отсортированный порядок**: элементы всегда хранятся в отсортированном порядке
- **Вероятностная структура**: использует случайность для балансировки
- **Потокобезопасность**: встроенная синхронизация для многопоточной работы

## Сравнение с другими структурами

Skip List находится между простым связанным списком и сбалансированным деревом поиска. Он обеспечивает хороший баланс между простотой реализации и производительностью поиска.
