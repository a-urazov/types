# Куча (Heap)

Этот пакет предоставляет обобщенную и поточно-безопасную реализацию структуры данных кучи.

## Особенности

-   **Обобщенная (Generic):** Может работать с любым типом данных.
-   **Настраиваемый порядок:** Порядок элементов (минимальная или максимальная куча) определяется с помощью функции-компаратора.
-   **Поточно-безопасная:** Безопасна для использования в нескольких горутинах.
-   **Основана на `container/heap`:** Использует стандартную библиотеку Go для реализации кучи.

## Использование

### Создание кучи

Для создания новой кучи используйте функцию `New`, передав ей компаратор.

```go
import "github.com/a-urazov/types/collections/heap"

// Создание минимальной кучи для int
minHeap := heap.New[int](func(a, b int) bool { return a < b })

// Создание максимальной кучи для int
maxHeap := heap.New[int](func(a, b int) bool { return a > b })
```

### Основные операции

```go
h := heap.New[int](func(a, b int) bool { return a < b })

// Добавление элементов
h.Push(10)
h.Push(20)
h.Push(5)

// Размер кучи
size := h.Size() // 3

// Просмотр верхнего элемента без удаления
top, ok := h.Peek() // 5, true

// Извлечение верхнего элемента
val, ok := h.Pop() // 5, true

// Проверка на пустоту
isEmpty := h.IsEmpty() // false
```
