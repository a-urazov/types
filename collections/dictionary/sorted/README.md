# Sorted Dictionary

Пакет `sorted` предоставляет реализацию отсортированного словаря, который хранит пары ключ-значение в отсортированном порядке ключей. Это позволяет итерировать по элементам в предсказуемом порядке.

## Основные методы

- `New[TKey, TValue]() *Dictionary[TKey, TValue]` - создает новый пустой отсортированный словарь
- `Set(key TKey, value TValue)` - добавляет или обновляет пару ключ-значение
- `Get(key TKey) (TValue, bool)` - получает значение по ключу
- `Remove(key TKey) bool` - удаляет пару по ключу
- `Contains(key TKey) bool` - проверяет наличие ключа
- `Size() int` - возвращает количество пар
- `IsEmpty() bool` - проверяет, пуст ли словарь
- `Clear()` - очищает словарь
- `Keys() []TKey` - возвращает все ключи в отсортированном порядке
- `Values() []TValue` - возвращает все значения в порядке их ключей
- `ForEach(fn func(key TKey, value TValue))` - итерирует по всем парам в отсортированном порядке

## Пример использования

```go
package main

import (
    "fmt"
    "types/collections/dictionary/sorted"
)

func main() {
    dict := sorted.New[string, int]()
    
    dict.Set("zebra", 26)
    dict.Set("apple", 1)
    dict.Set("banana", 2)
    
    // Итерирует в отсортированном порядке: apple, banana, zebra
    dict.ForEach(func(key string, val int) {
        fmt.Printf("%s: %d\n", key, val)
    })
    
    keys := dict.Keys()
    fmt.Println("Keys:", keys) // [apple banana zebra]
}
```

## Особенности

- **Отсортированный порядок**: ключи всегда хранятся в отсортированном порядке
- **Предсказуемая итерация**: элементы всегда обходятся в порядке ключей
- **Потокобезопасность**: встроенная синхронизация
- **Производительность**: O(log n) для поиска, вставки и удаления

## Применение

- Сортированное отображение данных
- Лексикографический порядок элементов
- Поддержание упорядоченного кэша
- Системы, где важен порядок обхода ключей
- Создание отсортированных индексов
