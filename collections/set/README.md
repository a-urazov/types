# Множество

Пакет `set` предоставляет реализацию набора уникальных элементов. Множество (Set) - это коллекция, которая содержит только уникальные элементы без дублирования.

## Основные методы

- `New[T]() *Set[T]` - создает новое пустое множество
- `Add(item T) bool` - добавляет элемент; возвращает false, если элемент уже существует
- `Remove(item T) bool` - удаляет элемент; возвращает false, если элемента нет
- `Contains(item T) bool` - проверяет наличие элемента
- `Size() int` - возвращает количество элементов
- `IsEmpty() bool` - проверяет, пусто ли множество
- `Clear()` - очищает множество
- `ToSlice() []T` - преобразует множество в срез
- `Union(other *Set[T]) *Set[T]` - возвращает объединение двух множеств
- `Intersection(other *Set[T]) *Set[T]` - возвращает пересечение двух множеств
- `Difference(other *Set[T]) *Set[T]` - возвращает разность двух множеств

## Пример использования

```go
package main

import (
    "fmt"
    "types/collections/set"
)

func main() {
    s := set.New[int]()

    s.Add(1)
    s.Add(2)
    s.Add(3)
    s.Add(2) // Не добавляется, так как уже существует

    if s.Contains(2) {
        fmt.Println("2 is in the set")
    }

    fmt.Println("Size:", s.Size())

    s2 := set.New[int]()
    s2.Add(2)
    s2.Add(3)
    s2.Add(4)

    union := s.Union(s2)
    fmt.Println("Union size:", union.Size())
}
```

## Особенности

- **Уникальность**: гарантирует, что каждый элемент появляется только один раз
- **Потокобезопасность**: встроенная синхронизация
- **Теоретико-множественные операции**: поддерживает объединение, пересечение, разность
- **Эффективность**: использует карту для O(1) операций в среднем

## Применение

- Удаление дубликатов из коллекции
- Проверка уникальности элементов
- Операции над множествами (объединение, пересечение, разность)
- Отслеживание посещенных узлов в графах
- Анализ данных и статистика
